# Contributing to Shokunin Static Site Generator (SSG)

Thank you for your interest in contributing to Shokunin. Whether you want to spread the word, share feedback, or contribute code, your involvement makes Shokunin better for everyone.

## Spread the word

One of the easiest ways to help is by telling others about Shokunin. A larger, more engaged community strengthens the project. If you know people who could benefit from Shokunin, let them know.

## How to contribute

You can contribute in several ways: sharing feedback, reporting bugs, or submitting code.

### Feedback

Your feedback helps us improve Shokunin. If you have ideas, suggestions, or questions, here is how to share them:

1. Open a [new feedback issue][2].
2. Use a descriptive title that clearly summarizes your feedback.
3. Provide a detailed description of your idea or suggestion.
4. Allow time for the team to review and respond.

### Bug reports

If you find a bug, report it so we can fix it:

1. Open a [new issue][2].
2. Use a descriptive title that clearly summarizes the bug.
3. Describe the issue in detail, including steps to reproduce it.
4. Allow time for the team to review and respond.

### Code contributions

We welcome code contributions. Here is what you need to know.

#### Feature requests

If you have an idea for a new feature or improvement:

1. Fork the repository.
2. Clone the Shokunin repo:
   `git clone https://github.com/sebastienrousseau/shokunin`
3. Make your changes in the `src/` folder, which contains the Shokunin source code.
4. Submit a pull request. We will review and merge changes that align with the project vision.

#### Submitting code

If you have identified a bug fix or a specific code improvement:

1. Fork the repository.
2. Clone the Shokunin repo:
   `git clone https://github.com/sebastienrousseau/shokunin`
3. Make your changes in the `src/` folder, which contains the Shokunin source code.
4. Submit a pull request. We will review and merge changes that align with the project vision.

Thank you for your interest and involvement in Shokunin.
