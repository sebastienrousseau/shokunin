Function: run - Runs the static site generator command-line tool.

This function prints a banner containing the title and description of the tool, and then processes any command-line arguments passed to it. If no arguments are passed, it prints a welcome message and instructions on how to use the tool.

The function uses the build function from the cli module to create the command-line interface for the tool. It then processes any arguments passed to it using the parser function from the args module.

If any errors occur during the process (e.g. an invalid argument is passed), an error message is printed and returned. Otherwise, Ok(()) is returned.
