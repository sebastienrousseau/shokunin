Function: build - returns a Result containing the parsed input options

Builds a command-line interface (CLI) using the clap crate and returns the command-line arguments passed to the CLI as an ArgMatches object.

This function creates a CLI, and adds the following arguments to it:

  • --new or -n: Creates a new project.
  • --content or -c: Specifies the location of the content directory.
  • --output or -o: Specifies the location of the output directory.
  • --template or -t: Specifies the location of the template directory.
  • --serve or -s: Serves the public directory on a local web server.

If the CLI is successfully built and the command-line arguments are parsed correctly, the function returns an Ok result containing the ArgMatches object. If an error occurs while building or parsing the CLI, an Err result containing the error message is returned.

Arguments

None

Returns

  • Result<ArgMatches, Error> - A struct containing the parsed command-line arguments and their values, or an error if the arguments could not be parsed.
