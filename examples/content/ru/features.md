## Стартовые шаблоны

Все стартовые шаблоны, будь то страницы или сайты, создаются с нуля
с использованием одних и тех же принципов и возможностей. Это означает,
что вы можете легко комбинировать их для создания собственных макетов и
контента.

Начните работу с SSG, используя любой из наших стартовых шаблонов для
создания своего сайта или веб-приложения.

**Стартовые шаблоны Kaishi** включают следующие возможности:

- **Метатеги доступности:** Эти метатеги предназначены для того, чтобы сделать сайт более доступным для пользователей с ограниченными возможностями. Установив роли и атрибуты Accessible Rich Internet Applications (ARIA), полный контроль с клавиатуры и отсутствие мигающих элементов, вы можете обеспечить доступность своего сайта для всех.
- **Метатеги Apple:** Эти метатеги улучшают отображение сайтов на устройствах Apple, таких как iPhone, iPad и других устройствах Apple. Вы можете изменить возможности веб-приложения, стиль строки состояния, заголовок, название приложения и автора, чтобы улучшить внешний вид на устройствах Apple.
- **Bootstrap CSS:** Bootstrap — это популярный CSS-фреймворк, который предоставляет набор готовых стилей и компонентов. Используя Bootstrap, вы можете быстро и легко создать профессионально выглядящий сайт, не написав CSS с нуля.
- **Bootstrap JavaScript:** Bootstrap JavaScript — это набор готовых скриптов, которые предоставляют адаптивные меню навигации и модальные окна.
- **Политика безопасности контента:** Этот метатег используется для указания источников контента, которые разрешено загружать на странице. Он предназначен для предотвращения атак межсайтового скриптинга (XSS) и других уязвимостей безопасности.
- **Метатеги Microsoft:** Эти метатеги предназначены для оптимизации сайта для устройств Microsoft. Вы можете установить верификацию сайта, конфигурацию приложения, цвет подсветки касания, цвет плитки и изображение плитки, чтобы сайт хорошо выглядел на устройствах Windows.
- **Метатеги Open Graph/Facebook:** Эти метатеги позволяют вам управлять тем, как ваш сайт отображается при публикации в Facebook и других социальных сетях. Установив заголовок, описание и изображение, вы можете убедиться, что ваш сайт выглядит наилучшим образом при публикации в интернете.
- **Адаптивная панель навигации:** Адаптивная панель навигации предоставляет пользователям интуитивный и простой в использовании интерфейс для навигации по сайту. Она адаптируется к размеру экрана, делая сайт доступным для пользователей как на настольных компьютерах, так и на мобильных устройствах.
- **Метатеги Schema.org:** Эти метатеги используются для предоставления структурированных данных о содержимом сайта. Установка имени, описания и изображения на сайте помогает поисковым системам и другим сервисам лучше понимать контент.
- **Метатеги Twitter:** Эти метатеги предназначены для оптимизации сайта для публикации в Twitter. Вы можете установить тип карточки, создателя, описание, изображение, сайт, заголовок и URL, чтобы ваш сайт хорошо выглядел в Twitter.
