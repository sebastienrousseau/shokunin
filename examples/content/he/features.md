## תבניות התחלה

כל תבניות ההתחלה, בין אם הן עמודים או אתרים שלמים, נבנות מאפס תוך שימוש
באותם עקרונות ותכונות. משמעות הדבר היא שניתן לערבב אותן בקלות
וליצור פריסות ותכנים מותאמים אישית משלכם.

התחילו עם SSG באמצעות אחת מתבניות ההתחלה שלנו לבניית
האתר או יישום האינטרנט שלכם.

**תבניות ההתחלה של Kaishi** כוללות את התכונות הבאות:

- **תגיות מטא לנגישות:** תגיות מטא אלו נועדו להפוך את האתר לנגיש יותר עבור משתמשים בעלי מוגבלויות. באמצעות הגדרת תפקידי ותכונות ARIA, שליטה מלאה במקלדת וללא סכנת הבהובים, תוכלו להבטיח שהאתר שלכם יהיה נגיש לכולם.
- **תגיות מטא של Apple:** תגיות מטא אלו משפרות את האתר עבור מכשירי Apple, כמו iPhone, iPad ומכשירי Apple אחרים. ניתן לשנות את יכולות אפליקציית האינטרנט, סגנון שורת המצב, הכותרת, שם האפליקציה והמחבר כדי לשפר את המראה במכשירי Apple.
- **Bootstrap CSS:** Bootstrap היא מסגרת CSS פופולרית המספקת סט של סגנונות ורכיבים מעוצבים מראש. באמצעות Bootstrap, ניתן ליצור במהירות ובקלות אתר בעל מראה מקצועי ללא צורך לכתוב CSS מאפס.
- **Bootstrap JavaScript:** Bootstrap JavaScript הוא אוסף של סקריפטים מוכנים מראש המספקים תפריטי ניווט רספונסיביים וחלונות דו-שיח מודאליים.
- **מדיניות אבטחת תוכן:** תגית מטא זו משמשת לציון מקורות התוכן המורשים להיטען בעמוד. היא נועדה למנוע התקפות Cross-Site Scripting (XSS) ופרצות אבטחה אחרות.
- **תגיות מטא של Microsoft:** תגיות מטא אלו נועדו להתאים את האתר עבור מכשירי Microsoft. ניתן להגדיר אימות אתר, תצורת אפליקציה, צבע הדגשת הקשה, צבע אריח ותמונת אריח כדי להציג מראה טוב במכשירי Windows.
- **תגיות מטא של Open Graph/Facebook:** תגיות מטא אלו מאפשרות לכם לשלוט בצורת ההופעה של האתר שלכם בעת שיתופו בפייסבוק וברשתות חברתיות אחרות. באמצעות הגדרת הכותרת, התיאור והתמונה, תוכלו להבטיח שהאתר שלכם ייראה במיטבו בעת השיתוף ברשת.
- **סרגל ניווט רספונסיבי:** סרגל הניווט הרספונסיבי מספק למשתמשים ממשק אינטואיטיבי וקל לשימוש לניווט באתר. הוא מותאם לגודל המסך, ובכך נגיש למשתמשים במחשבים שולחניים ובמכשירים ניידים כאחד.
- **תגיות מטא של Schema.org:** תגיות מטא אלו משמשות לאספקת נתונים מובנים על תוכן האתר. הגדרת השם, התיאור והתמונה באתר עוזרת למנועי חיפוש ולאחרים להבין את התוכן טוב יותר.
- **תגיות מטא של Twitter:** תגיות מטא אלו נועדו להתאים את האתר לשיתוף בטוויטר. ניתן להגדיר את סוג הכרטיס, היוצר, התיאור, התמונה, האתר, הכותרת והכתובת כדי שהאתר ייראה טוב בטוויטר.
