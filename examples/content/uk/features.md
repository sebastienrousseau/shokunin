## Стартові шаблони

Усі стартові шаблони — чи то сторінки, чи то цілі сайти — створено з нуля
за одними й тими ж принципами та можливостями. Це означає, що ви можете легко
поєднувати їх між собою, створюючи власні макети та зміст.

Почніть роботу з SSG, використовуючи будь-який із наших стартових шаблонів
для створення вашого сайту або вебзастосунку.

У **стартових шаблонах Kaishi** увімкнено такі функції:

- **Метатеги доступності:** Ці метатеги створено для того, щоб зробити сайт доступнішим для користувачів з інвалідністю. Задаючи ролі та атрибути ARIA (Accessible Rich Internet Applications), повне керування з клавіатури та відсутність мерехтіння, ви можете забезпечити доступність вашого сайту для всіх.
- **Метатеги Apple:** Ці метатеги покращують сайти для пристроїв Apple, таких як iPhone, iPad та інших пристроїв Apple. Ви можете змінювати можливості вебзастосунку, стиль рядка стану, заголовок, назву застосунку та автора, щоб покращити вигляд на пристроях Apple.
- **Bootstrap CSS:** Bootstrap — популярний CSS-фреймворк, який надає набір попередньо розроблених стилів та компонентів. Використовуючи Bootstrap, ви можете швидко й легко створити сайт професійного вигляду, не пишучи CSS з нуля.
- **Bootstrap JavaScript:** Bootstrap JavaScript — це набір готових скриптів, які надають адаптивні навігаційні меню та модальні діалоги.
- **Політика безпеки контенту:** Цей метатег використовується для визначення джерел контенту, які дозволено завантажувати на сторінці. Його призначення — запобігати атакам міжсайтового скриптингу (XSS) та іншим вразливостям безпеки.
- **Метатеги Microsoft:** Ці метатеги призначені для оптимізації сайту для пристроїв Microsoft. Ви можете задати перевірку сайту, конфігурацію застосунку, колір підсвічування, колір плитки та зображення плитки, щоб сайт мав гарний вигляд на пристроях Windows.
- **Метатеги Open Graph/Facebook:** Ці метатеги дозволяють контролювати, як ваш сайт відображається під час поширення у Facebook та інших соціальних мережах. Задаючи заголовок, опис та зображення, ви можете переконатися, що ваш сайт має найкращий вигляд під час поширення онлайн.
- **Адаптивна навігаційна панель:** Адаптивна навігаційна панель надає користувачам інтуїтивно зрозумілий і простий у використанні інтерфейс для навігації сайтом. Вона пристосовується до розміру екрана, роблячи сайт доступним для користувачів як на настільних комп'ютерах, так і на мобільних пристроях.
- **Метатеги Schema.org:** Ці метатеги використовуються для надання структурованих даних про зміст сайту. Задання імені, опису та зображення на сайті допомагає пошуковим системам та іншим сервісам краще розуміти зміст.
- **Метатеги Twitter:** Ці метатеги призначені для оптимізації сайту для поширення у Twitter. Ви можете задати тип картки, творця, опис, зображення, сайт, заголовок та URL, щоб сайт добре виглядав у Twitter.
