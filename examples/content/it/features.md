## Template di partenza

Tutti i template di partenza, sia per pagine che per siti web, sono costruiti
da zero utilizzando gli stessi principi e le stesse funzionalità. Questo
significa che puoi facilmente combinarli tra loro per creare i tuoi layout e
contenuti personalizzati.

Inizia con SSG utilizzando uno qualsiasi dei nostri template di partenza per
costruire il tuo sito web o la tua applicazione web.

I **Template di partenza Kaishi** includono le seguenti funzionalità:

- **Meta tag di accessibilità:** Questi meta tag sono progettati per rendere il sito web più accessibile agli utenti con disabilità. Impostando i ruoli e gli attributi ARIA (Accessible Rich Internet Applications), il pieno controllo da tastiera e l'assenza di lampeggi pericolosi, puoi assicurarti che il tuo sito web sia accessibile a tutti.
- **Meta tag Apple:** Questi meta tag migliorano i siti web per i dispositivi Apple, come iPhone, iPad e dispositivi Apple. Puoi modificare le capacità della web app, lo stile della barra di stato, il titolo, il nome dell'applicazione e l'autore per migliorarne l'aspetto sui dispositivi Apple.
- **Bootstrap CSS:** Bootstrap è un popolare framework CSS che ti fornisce un set di stili e componenti predefiniti. Usando Bootstrap puoi creare rapidamente e facilmente un sito web dall'aspetto professionale senza dover scrivere il CSS da zero.
- **Bootstrap JavaScript:** Bootstrap JavaScript è un insieme di script predefiniti che ti fornisce menu di navigazione responsive e finestre di dialogo modali.
- **Content Security Policy:** Questo meta tag viene utilizzato per specificare le origini dei contenuti che possono essere caricati sulla pagina. È progettato per prevenire attacchi di cross-site scripting (XSS) e altre vulnerabilità di sicurezza.
- **Meta tag Microsoft:** Questi meta tag sono progettati per ottimizzare il sito web per i dispositivi Microsoft. Puoi impostare la verifica del sito, la configurazione dell'applicazione, il colore di evidenziazione del tocco, il colore della tile e l'immagine della tile per rendere il sito esteticamente piacevole sui dispositivi Windows.
- **Meta tag Open Graph/Facebook:** Questi meta tag ti consentono di controllare come appare il tuo sito web quando viene condiviso su Facebook e altre piattaforme social. Impostando titolo, descrizione e immagine puoi assicurarti che il tuo sito appaia al meglio quando viene condiviso online.
- **Barra di navigazione responsive:** La barra di navigazione responsive offre agli utenti un'interfaccia intuitiva e facile da usare per navigare nel sito. Si adatta alle dimensioni dello schermo, rendendola accessibile sia agli utenti desktop che a quelli mobili.
- **Meta tag Schema.org:** Questi meta tag vengono utilizzati per fornire dati strutturati sui contenuti del sito. Impostare nome, descrizione e immagine aiuta i motori di ricerca e gli altri a comprenderne meglio il contenuto.
- **Meta tag Twitter:** Questi meta tag sono progettati per ottimizzare il sito web per la condivisione su Twitter. Puoi impostare il tipo di card, il creatore, la descrizione, l'immagine, il sito, il titolo e l'URL per far sì che il sito appaia al meglio su Twitter.
