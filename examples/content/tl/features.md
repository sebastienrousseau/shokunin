## Mga Starter Template

Ang lahat ng starter templates, ito man ay mga pahina o website, ay
itinatayo mula sa simula gamit ang parehong mga prinsipyo at tampok. Nangangahulugan
ito na madali mong mahaluan at maitutugma ang mga ito upang lumikha ng iyong
sariling custom na layout at nilalaman.

Magsimula sa SSG gamit ang alinman sa aming mga Starter Template para sa
pagbuo ng iyong website o web application.

Ang **Kaishi Starter Templates** ay may mga sumusunod na tampok na pinagana:

- **Mga Accessibility Meta-Tag:** Ang mga meta tag na ito ay idinisenyo upang gawing mas naa-access ang website sa mga gumagamit na may kapansanan. Sa pamamagitan ng pagtatakda ng mga Accessible Rich Internet Applications (ARIA) na role at attribute, buong kontrol ng keyboard, at walang panganib sa pag-flash, masisigurado mong naa-access ang iyong website sa lahat.
- **Mga Apple Meta-Tag:** Pinapabuti ng mga meta tag na ito ang mga website para sa mga Apple device, tulad ng iPhone, iPad, at mga Apple device. Maaari mong baguhin ang web app capabilities, status bar style, title, application name, at author upang mapabuti ang hitsura ng mga Apple device.
- **Bootstrap CSS:** Ang Bootstrap ay isang sikat na CSS framework na nagbibigay sa iyo ng isang hanay ng mga paunang dinisenyong istilo at component. Sa paggamit ng Bootstrap, maaari kang mabilis at madaling lumikha ng isang propesyonal na hitsurang website nang hindi kailangang magsulat ng CSS mula sa simula.
- **Bootstrap JavaScript:** Ang Bootstrap JavaScript ay isang hanay ng mga paunang itinayong script na nagbibigay sa iyo ng mga responsive na navigation menu at modal dialog.
- **Content Security Policy:** Ang meta tag na ito ay ginagamit upang tukuyin ang mga pinagmulan ng nilalamang pinapayagang i-load sa pahina. Ito ay idinisenyo upang maiwasan ang mga pag-atake ng cross-site scripting (XSS) at iba pang kahinaan sa seguridad.
- **Mga Microsoft Meta Tag:** Ang mga meta tag na ito ay idinisenyo upang i-optimize ang website para sa mga Microsoft device. Maaari mong itakda ang site verification, application configuration, tap highlight color, tile color, at tile image upang magmukhang maganda sa mga Windows device.
- **Mga Open Graph/Facebook Meta-Tag:** Ang mga meta tag na ito ay nagpapahintulot sa iyong kontrolin kung paano lumalabas ang iyong website kapag ibinahagi sa Facebook at iba pang social media platform. Sa pamamagitan ng pagtatakda ng pamagat, paglalarawan, at imahe, masisigurado mong magmumukha nang pinakamahusay ang iyong website kapag ibinahagi online.
- **Responsive na Navigation Bar:** Ang responsive na navigation bar ay nagbibigay sa mga gumagamit ng intuitive at madaling gamiting interface para sa pag-navigate sa website. Layunin nitong umangkop sa laki ng screen, na ginagawang naa-access ito sa mga gumagamit sa parehong desktop at mobile device.
- **Mga Schema.org Meta Tag:** Ang mga meta tag na ito ay ginagamit upang magbigay ng nakabalangkas na data tungkol sa nilalaman ng website. Ang pagtatakda ng pangalan, paglalarawan, at imahe sa isang website ay tumutulong sa mga search engine at iba pa upang mas maintindihan ang nilalaman.
- **Mga Twitter Meta Tag:** Ang mga meta tag na ito ay idinisenyo upang i-optimize ang website para sa pagbabahagi sa Twitter. Maaari mong itakda ang uri ng card, lumikha, paglalarawan, imahe, site, pamagat, at URL upang magmukhang maganda ang kanilang website sa Twitter.
