## Pangkalahatang-ideya

Ang **Kaishi** ay isang minimalist at modernong [SSG static site generator ⧉][0]
starter template na idinisenyo para sa mga propesyonal na nagpapahalaga sa
pagiging simple at eleganteng disenyo.

Sa malinis at dinamikong layout nito, nag-aalok ang Kaishi ng maraming gamit
at madaling gamiting solusyon para sa mga naghahanap na ipakita ang kanilang
trabaho at serbisyo online. Itinayo sa isang responsive na framework, ang
template na ito ay angkop para sa mga propesyonal na walang kakayahan sa coding
o pagdidisenyo.

Maging ikaw man ay isang freelance na malikhain, isang tagapagtatag ng
startup, o isang may-ari ng maliit na negosyo. Ang handa-nang-gamitin na
website at responsive na starter templates ng Kaishi ay nagbibigay ng
perpektong pundasyon para sa iyong presensya online. Sa minimalist na disenyo
nito, ang Kaishi ang pinakamahusay na website starter template para sa
modernong at propesyonal na mga website.

Ang pahinang ito ay isang halimbawa para sa SSG static site generator. Maaari
mo itong gamiting template para sa iyong website o blog. Gumagamit ito ng
markdown template para sa nilalaman at custom HTML theme para sa layout.
