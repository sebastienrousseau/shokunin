## スターターテンプレート

ページでもウェブサイトでも、すべてのスターターテンプレートは同じ原則と機能を用いてゼロから構築されています。そのため、簡単に組み合わせて、独自のカスタムレイアウトやコンテンツを作成できます。

ウェブサイトやウェブアプリケーションの構築には、当社のスターターテンプレートのいずれかを使って SSG を始めましょう。

**Kaishi スターターテンプレート** には以下の機能が搭載されています。

- **アクセシビリティメタタグ:** これらのメタタグは、障害のあるユーザーにとってウェブサイトをより利用しやすくするために設計されています。Accessible Rich Internet Applications (ARIA) のロールと属性、フルキーボード操作、点滅ハザードなしの設定により、誰もがアクセスできるウェブサイトを実現できます。
- **Apple メタタグ:** これらのメタタグは、iPhone、iPad などの Apple デバイス向けにウェブサイトを最適化します。ウェブアプリの機能、ステータスバーのスタイル、タイトル、アプリケーション名、作者を変更して、Apple デバイスでの見た目を改善できます。
- **Bootstrap CSS:** Bootstrap は人気のある CSS フレームワークで、事前に設計されたスタイルとコンポーネントのセットを提供します。Bootstrap を使用すれば、CSS をゼロから書くことなく、プロフェッショナルなウェブサイトを素早く簡単に作成できます。
- **Bootstrap JavaScript:** Bootstrap JavaScript は、レスポンシブなナビゲーションメニューやモーダルダイアログを提供するビルド済みスクリプトのセットです。
- **コンテンツセキュリティポリシー:** このメタタグは、ページに読み込める許可コンテンツのソースを指定するために使用されます。クロスサイトスクリプティング (XSS) 攻撃やその他のセキュリティ脆弱性を防ぐために設計されています。
- **Microsoft メタタグ:** これらのメタタグは、Microsoft デバイス向けにウェブサイトを最適化するために設計されています。サイト検証、アプリケーション構成、タップハイライトの色、タイルの色、タイル画像を設定して、Windows デバイスでの見栄えを良くすることができます。
- **Open Graph / Facebook メタタグ:** これらのメタタグは、Facebook などのソーシャルメディアで共有された際のウェブサイトの表示方法を制御します。タイトル、説明、画像を設定することで、オンラインで共有された際にウェブサイトが最高の状態で表示されることを保証できます。
- **レスポンシブナビゲーションバー:** レスポンシブナビゲーションバーは、ウェブサイトをナビゲートするための直感的で使いやすいインターフェースをユーザーに提供します。画面サイズに合わせて適応し、デスクトップとモバイルの両方のデバイスでアクセスしやすくすることを目指しています。
- **Schema.org メタタグ:** これらのメタタグは、ウェブサイトのコンテンツに関する構造化データを提供するために使用されます。ウェブサイトに名前、説明、画像を設定することで、検索エンジンなどがコンテンツをよりよく理解できるようになります。
- **Twitter メタタグ:** これらのメタタグは、Twitter での共有向けにウェブサイトを最適化するために設計されています。カードの種類、作成者、説明、画像、サイト、タイトル、URL を設定して、Twitter 上でウェブサイトが良く見えるようにできます。
