## Templat Awal

Semua templat awal, baik halaman maupun situs web, dibangun dari awal menggunakan prinsip dan fitur yang sama. Ini berarti Anda dapat dengan mudah memadukan dan mencocokkannya untuk membuat tata letak dan konten kustom Anda sendiri.

Mulailah dengan SSG menggunakan salah satu Templat Awal kami untuk membangun situs web atau aplikasi web Anda.

**Templat Awal Kaishi** memiliki fitur-fitur berikut yang diaktifkan:

- **Tag Meta Aksesibilitas:** Tag meta ini dirancang untuk membuat situs web lebih mudah diakses oleh pengguna dengan disabilitas. Dengan menetapkan peran dan atribut Accessible Rich Internet Applications (ARIA), kontrol keyboard penuh, dan tanpa bahaya kedipan, Anda dapat memastikan situs web Anda dapat diakses oleh semua orang.
- **Tag Meta Apple:** Tag meta ini meningkatkan situs web untuk perangkat Apple, seperti iPhone, iPad, dan perangkat Apple. Anda dapat mengubah kemampuan aplikasi web, gaya bilah status, judul, nama aplikasi, dan penulis untuk meningkatkan tampilan perangkat Apple.
- **Bootstrap CSS:** Bootstrap adalah kerangka kerja CSS populer yang memberi Anda satu set gaya dan komponen yang telah dirancang sebelumnya. Dengan menggunakan Bootstrap, Anda dapat dengan cepat dan mudah membuat situs web yang tampak profesional tanpa harus menulis CSS dari awal.
- **Bootstrap JavaScript:** Bootstrap JavaScript adalah satu set skrip yang telah dibangun sebelumnya yang memberi Anda menu navigasi responsif dan dialog modal.
- **Content Security Policy:** Tag meta ini digunakan untuk menentukan sumber konten yang diizinkan untuk dimuat pada halaman. Ini dirancang untuk mencegah serangan cross-site scripting (XSS) dan kerentanan keamanan lainnya.
- **Tag Meta Microsoft:** Tag meta ini dirancang untuk mengoptimalkan situs web untuk perangkat Microsoft. Anda dapat menetapkan verifikasi situs, konfigurasi aplikasi, warna sorot ketuk, warna ubin, dan gambar ubin agar terlihat bagus pada perangkat Windows.
- **Tag Meta Open Graph/Facebook:** Tag meta ini memungkinkan Anda mengontrol bagaimana situs web Anda muncul saat dibagikan di Facebook dan platform media sosial lainnya. Dengan menetapkan judul, deskripsi, dan gambar, Anda dapat memastikan bahwa situs web Anda terlihat sebaik mungkin saat dibagikan secara online.
- **Bilah Navigasi Responsif:** Bilah navigasi responsif memberi pengguna antarmuka yang intuitif dan mudah digunakan untuk menavigasi situs web. Bertujuan untuk beradaptasi dengan ukuran layar, sehingga dapat diakses oleh pengguna pada perangkat desktop maupun seluler.
- **Tag Meta Schema.org:** Tag meta ini digunakan untuk menyediakan data terstruktur tentang konten situs web. Mengatur nama, deskripsi, dan gambar pada situs web membantu mesin pencari dan lainnya memahami konten dengan lebih baik.
- **Tag Meta Twitter:** Tag meta ini dirancang untuk mengoptimalkan situs web untuk berbagi di Twitter. Anda dapat menetapkan tipe kartu, pembuat, deskripsi, gambar, situs, judul, dan URL agar situs web mereka terlihat bagus di Twitter.
