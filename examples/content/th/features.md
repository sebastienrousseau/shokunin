## เทมเพลตเริ่มต้น

เทมเพลตเริ่มต้นทั้งหมด ไม่ว่าจะเป็นหน้าหรือเว็บไซต์ ล้วนสร้างขึ้นใหม่
โดยใช้หลักการและคุณสมบัติเดียวกัน หมายความว่าคุณสามารถผสมผสานและ
จับคู่กันได้อย่างง่ายดายเพื่อสร้างเลย์เอาต์และเนื้อหาที่กำหนดเอง

เริ่มต้นใช้งาน SSG ด้วยเทมเพลตเริ่มต้นของเราเพื่อสร้างเว็บไซต์
หรือเว็บแอปพลิเคชันของคุณ

**เทมเพลตเริ่มต้น Kaishi** มีคุณสมบัติดังต่อไปนี้:

- **Meta-Tags การเข้าถึง:** Meta tags เหล่านี้ออกแบบมาเพื่อทำให้เว็บไซต์เข้าถึงได้ง่ายขึ้นสำหรับผู้ใช้ที่มีความพิการ ด้วยการกำหนดบทบาทและคุณลักษณะ Accessible Rich Internet Applications (ARIA) การควบคุมแป้นพิมพ์เต็มรูปแบบ และไม่มีอันตรายจากแสงกะพริบ คุณจึงมั่นใจได้ว่าเว็บไซต์ของคุณเข้าถึงได้สำหรับทุกคน
- **Apple Meta-Tags:** Meta tags เหล่านี้ปรับปรุงเว็บไซต์สำหรับอุปกรณ์ Apple เช่น iPhones, iPads และอุปกรณ์ Apple อื่นๆ คุณสามารถเปลี่ยนความสามารถของเว็บแอป รูปแบบแถบสถานะ ชื่อ ชื่อแอปพลิเคชัน และผู้เขียน เพื่อปรับปรุงรูปลักษณ์บนอุปกรณ์ Apple
- **Bootstrap CSS:** Bootstrap เป็นเฟรมเวิร์ก CSS ยอดนิยมที่ให้ชุดสไตล์และส่วนประกอบที่ออกแบบไว้ล่วงหน้า การใช้ Bootstrap ช่วยให้คุณสร้างเว็บไซต์ที่ดูเป็นมืออาชีพได้อย่างรวดเร็วและง่ายดาย โดยไม่ต้องเขียน CSS ใหม่ตั้งแต่ต้น
- **Bootstrap JavaScript:** Bootstrap JavaScript คือชุดสคริปต์ที่สร้างไว้ล่วงหน้าซึ่งให้เมนูนำทางที่ตอบสนองและกล่องโต้ตอบโมดอล
- **Content Security Policy:** Meta tag นี้ใช้เพื่อระบุแหล่งที่มาของเนื้อหาที่อนุญาตให้โหลดบนหน้า ออกแบบมาเพื่อป้องกันการโจมตี cross-site scripting (XSS) และช่องโหว่ด้านความปลอดภัยอื่นๆ
- **Microsoft Meta Tags:** Meta tags เหล่านี้ออกแบบมาเพื่อปรับเว็บไซต์ให้เหมาะสมสำหรับอุปกรณ์ Microsoft คุณสามารถตั้งค่าการยืนยันเว็บไซต์ การกำหนดค่าแอปพลิเคชัน สีไฮไลต์การแตะ สีไทล์ และภาพไทล์ เพื่อให้ดูดีบนอุปกรณ์ Windows
- **Open Graph/Facebook Meta-Tags:** Meta tags เหล่านี้อนุญาตให้คุณควบคุมว่าเว็บไซต์ของคุณจะแสดงอย่างไรเมื่อแชร์บน Facebook และแพลตฟอร์มโซเชียลมีเดียอื่นๆ ด้วยการตั้งชื่อเรื่อง คำอธิบาย และภาพ คุณสามารถมั่นใจได้ว่าเว็บไซต์ของคุณจะดูดีที่สุดเมื่อแชร์ออนไลน์
- **แถบนำทางที่รองรับทุกอุปกรณ์:** แถบนำทางที่รองรับทุกอุปกรณ์ให้อินเทอร์เฟซที่ใช้งานง่ายและเป็นธรรมชาติแก่ผู้ใช้ในการนำทางเว็บไซต์ มุ่งเน้นการปรับให้เข้ากับขนาดหน้าจอ ทำให้เข้าถึงได้ทั้งบนเดสก์ท็อปและอุปกรณ์มือถือ
- **Schema.org Meta Tags:** Meta tags เหล่านี้ใช้เพื่อให้ข้อมูลที่มีโครงสร้างเกี่ยวกับเนื้อหาของเว็บไซต์ การตั้งชื่อ คำอธิบาย และภาพบนเว็บไซต์ช่วยให้เครื่องมือค้นหาและผู้อื่นเข้าใจเนื้อหาได้ดีขึ้น
- **Twitter Meta Tags:** Meta tags เหล่านี้ออกแบบมาเพื่อปรับเว็บไซต์ให้เหมาะสมสำหรับการแชร์บน Twitter คุณสามารถตั้งประเภทการ์ด ผู้สร้าง คำอธิบาย ภาพ เว็บไซต์ ชื่อเรื่อง และ URL เพื่อทำให้เว็บไซต์ของพวกเขาดูดีบน Twitter
