## Startmallar

Alla startmallar, oavsett om det är sidor eller webbplatser, är byggda från
grunden enligt samma principer och funktioner. Det betyder att du enkelt kan
mixa och matcha dem för att skapa egna anpassade layouter och innehåll.

Kom igång med SSG genom att använda någon av våra startmallar för att bygga
din webbplats eller webbapplikation.

**Kaishi startmallar** har följande funktioner aktiverade:

- **Tillgänglighetsmetataggar:** Dessa metataggar är utformade för att göra webbplatsen mer tillgänglig för användare med funktionsnedsättningar. Genom att ange Accessible Rich Internet Applications (ARIA)-roller och attribut, fullständig tangentbordskontroll och ingen risk för blinkande effekter kan du säkerställa att din webbplats är tillgänglig för alla.
- **Apple-metataggar:** Dessa metataggar förbättrar webbplatser för Apple-enheter, som iPhones, iPads och andra Apple-enheter. Du kan ändra webbappens funktioner, statusfältets stil, titel, programnamn och författare för att förbättra utseendet på Apple-enheter.
- **Bootstrap CSS:** Bootstrap är ett populärt CSS-ramverk som ger dig en uppsättning fördesignade stilar och komponenter. Med Bootstrap kan du snabbt och enkelt skapa en professionell webbplats utan att behöva skriva CSS från grunden.
- **Bootstrap JavaScript:** Bootstrap JavaScript är en uppsättning fördefinierade skript som ger dig responsiva navigeringsmenyer och modala dialoger.
- **Content Security Policy:** Denna metatagg används för att specificera vilka källor till innehåll som tillåts laddas på sidan. Den är utformad för att förhindra cross-site scripting (XSS)-attacker och andra säkerhetssårbarheter.
- **Microsoft-metataggar:** Dessa metataggar är utformade för att optimera webbplatsen för Microsoft-enheter. Du kan ange webbplatsverifiering, programkonfiguration, tap highlight-färg, tile-färg och tile-bild för att se bra ut på Windows-enheter.
- **Open Graph/Facebook-metataggar:** Dessa metataggar låter dig styra hur din webbplats visas när den delas på Facebook och andra sociala medier. Genom att ange titel, beskrivning och bild kan du säkerställa att din webbplats ser bäst ut när den delas online.
- **Responsiv navigeringsmeny:** Den responsiva navigeringsmenyn ger användarna ett intuitivt och lättanvänt gränssnitt för att navigera på webbplatsen. Den anpassar sig efter skärmens storlek så att den är tillgänglig för användare på både stationära och mobila enheter.
- **Schema.org-metataggar:** Dessa metataggar används för att tillhandahålla strukturerad data om webbplatsens innehåll. Att ange namn, beskrivning och bild på en webbplats hjälper sökmotorer och andra att förstå innehållet bättre.
- **Twitter-metataggar:** Dessa metataggar är utformade för att optimera webbplatsen för delning på Twitter. Du kan ange korttyp, skapare, beskrivning, bild, webbplats, titel och URL för att webbplatsen ska se bra ut på Twitter.
