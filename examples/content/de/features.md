## Starter-Vorlagen

Alle Starter-Vorlagen, ob Seiten oder ganze Websites, werden von Grund auf
nach denselben Prinzipien und Funktionen erstellt. Das bedeutet, dass Sie sie
problemlos miteinander kombinieren können, um Ihre eigenen individuellen
Layouts und Inhalte zu erstellen.

Starten Sie mit SSG mithilfe einer unserer Starter-Vorlagen, um Ihre Website
oder Webanwendung zu erstellen.

Die **Kaishi-Starter-Vorlagen** verfügen über die folgenden Funktionen:

- **Barrierefreiheits-Meta-Tags:** Diese Meta-Tags sind so konzipiert, dass die Website für Menschen mit Behinderungen besser zugänglich ist. Durch das Festlegen von ARIA-Rollen (Accessible Rich Internet Applications), vollständiger Tastatursteuerung und keinerlei Blinkgefahren stellen Sie sicher, dass Ihre Website für alle zugänglich ist.
- **Apple-Meta-Tags:** Diese Meta-Tags optimieren Websites für Apple-Geräte wie iPhones und iPads. Sie können Web-App-Funktionen, Statusleistenstil, Titel, Anwendungsnamen und Autor anpassen, um das Erscheinungsbild auf Apple-Geräten zu verbessern.
- **Bootstrap CSS:** Bootstrap ist ein beliebtes CSS-Framework, das Ihnen vorgefertigte Stile und Komponenten bereitstellt. Mit Bootstrap erstellen Sie schnell und einfach eine professionell wirkende Website, ohne CSS von Grund auf schreiben zu müssen.
- **Bootstrap JavaScript:** Bootstrap JavaScript ist eine Sammlung vorgefertigter Skripte, die responsive Navigationsmenüs und modale Dialoge bereitstellen.
- **Content Security Policy:** Dieses Meta-Tag legt fest, aus welchen Quellen Inhalte auf der Seite geladen werden dürfen. Es soll Cross-Site-Scripting (XSS)-Angriffe und andere Sicherheitslücken verhindern.
- **Microsoft-Meta-Tags:** Diese Meta-Tags optimieren die Website für Microsoft-Geräte. Sie können Site-Verifizierung, Anwendungskonfiguration, Tap-Highlight-Farbe, Tile-Farbe und Tile-Bild festlegen, damit die Website auf Windows-Geräten gut aussieht.
- **Open-Graph/Facebook-Meta-Tags:** Mit diesen Meta-Tags steuern Sie, wie Ihre Website beim Teilen auf Facebook und anderen sozialen Plattformen erscheint. Durch das Festlegen von Titel, Beschreibung und Bild stellen Sie sicher, dass Ihre Website beim Online-Teilen optimal aussieht.
- **Responsive Navigationsleiste:** Die responsive Navigationsleiste bietet Nutzern eine intuitive und einfach zu bedienende Oberfläche zum Navigieren auf der Website. Sie passt sich an die Bildschirmgröße an und ist sowohl auf Desktops als auch auf Mobilgeräten zugänglich.
- **Schema.org-Meta-Tags:** Diese Meta-Tags liefern strukturierte Daten zum Inhalt der Website. Das Festlegen von Name, Beschreibung und Bild hilft Suchmaschinen und anderen, den Inhalt besser zu verstehen.
- **Twitter-Meta-Tags:** Diese Meta-Tags optimieren die Website für das Teilen auf Twitter. Sie können Kartentyp, Ersteller, Beschreibung, Bild, Site, Titel und URL festlegen, damit die Website auf Twitter gut aussieht.
