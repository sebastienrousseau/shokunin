## Șabloane de bază

Toate șabloanele de bază, fie că sunt pagini sau site-uri web, sunt construite
de la zero pe baza acelorași principii și funcționalități. Acest lucru înseamnă
că le puteți combina cu ușurință pentru a vă crea propriile aspecte și conținuturi
personalizate.

Începeți cu SSG folosind oricare dintre șabloanele noastre de bază pentru a vă
construi site-ul web sau aplicația web.

**Șabloanele de bază Kaishi** au activate următoarele funcționalități:

- **Metaetichete de accesibilitate:** Aceste metaetichete sunt concepute pentru a face site-ul mai accesibil utilizatorilor cu dizabilități. Setând rolurile și atributele Accessible Rich Internet Applications (ARIA), controlul complet de la tastatură și absența pericolelor de pâlpâire, vă puteți asigura că site-ul este accesibil tuturor.
- **Metaetichete Apple:** Aceste metaetichete optimizează site-urile pentru dispozitivele Apple, precum iPhone, iPad și alte dispozitive Apple. Puteți modifica capacitățile aplicației web, stilul barei de stare, titlul, numele aplicației și autorul pentru a îmbunătăți aspectul pe dispozitivele Apple.
- **Bootstrap CSS:** Bootstrap este un cadru CSS popular care vă pune la dispoziție un set de stiluri și componente pre-proiectate. Folosind Bootstrap, puteți crea rapid și ușor un site cu aspect profesional fără a fi nevoie să scrieți CSS de la zero.
- **Bootstrap JavaScript:** Bootstrap JavaScript este un set de scripturi pregătite care vă oferă meniuri de navigare responsive și ferestre modale.
- **Politică de securitate a conținutului:** Această metaetichetă este utilizată pentru a specifica sursele de conținut permise să se încarce pe pagină. Este concepută pentru a preveni atacurile cross-site scripting (XSS) și alte vulnerabilități de securitate.
- **Metaetichete Microsoft:** Aceste metaetichete sunt concepute pentru a optimiza site-ul pentru dispozitivele Microsoft. Puteți seta verificarea site-ului, configurația aplicației, culoarea de evidențiere la atingere, culoarea și imaginea plăcii pentru ca site-ul să arate bine pe dispozitivele Windows.
- **Metaetichete Open Graph/Facebook:** Aceste metaetichete vă permit să controlați modul în care apare site-ul atunci când este distribuit pe Facebook și pe alte rețele sociale. Setând titlul, descrierea și imaginea, vă puteți asigura că site-ul arată cât mai bine atunci când este distribuit online.
- **Bară de navigare responsivă:** Bara de navigare responsivă oferă utilizatorilor o interfață intuitivă și ușor de folosit pentru navigarea pe site. Aceasta urmărește să se adapteze la dimensiunea ecranului, fiind accesibilă utilizatorilor atât pe desktop, cât și pe dispozitivele mobile.
- **Metaetichete Schema.org:** Aceste metaetichete sunt utilizate pentru a furniza date structurate despre conținutul site-ului. Setarea numelui, descrierii și imaginii pe un site ajută motoarele de căutare și altele să înțeleagă mai bine conținutul.
- **Metaetichete Twitter:** Aceste metaetichete sunt concepute pentru a optimiza site-ul pentru distribuirea pe Twitter. Puteți seta tipul cardului, creatorul, descrierea, imaginea, site-ul, titlul și URL-ul pentru ca site-ul să arate bine pe Twitter.
