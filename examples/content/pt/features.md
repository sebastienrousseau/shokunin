## Modelos iniciais

Todos os modelos iniciais, sejam páginas ou sítios, são construídos de raiz
seguindo os mesmos princípios e funcionalidades. Isto significa que pode facilmente
combiná-los para criar os seus próprios layouts e conteúdos personalizados.

Comece a usar o SSG com qualquer um dos nossos modelos iniciais para construir
o seu sítio ou aplicação web.

Os **modelos iniciais Kaishi** têm as seguintes funcionalidades ativadas:

- **Meta-tags de acessibilidade:** Estas meta tags foram concebidas para tornar o sítio mais acessível a utilizadores com deficiência. Ao definir funções e atributos ARIA (Accessible Rich Internet Applications), controlo total por teclado e ausência de risco de cintilação, pode garantir que o seu sítio é acessível a todos.
- **Meta-tags Apple:** Estas meta tags melhoram os sítios para dispositivos Apple, como iPhones, iPads e outros dispositivos Apple. Pode alterar as capacidades da aplicação web, o estilo da barra de estado, o título, o nome da aplicação e o autor para melhorar o aspeto nos dispositivos Apple.
- **Bootstrap CSS:** O Bootstrap é uma popular framework CSS que fornece um conjunto de estilos e componentes pré-desenhados. Ao usar o Bootstrap, pode criar rápida e facilmente um sítio com aspeto profissional sem ter de escrever CSS de raiz.
- **Bootstrap JavaScript:** O Bootstrap JavaScript é um conjunto de scripts pré-construídos que disponibilizam menus de navegação responsivos e diálogos modais.
- **Política de segurança de conteúdo:** Esta meta tag é usada para especificar as fontes de conteúdo permitidas a carregar na página. Foi concebida para prevenir ataques de cross-site scripting (XSS) e outras vulnerabilidades de segurança.
- **Meta tags Microsoft:** Estas meta tags foram concebidas para otimizar o sítio para dispositivos Microsoft. Pode definir a verificação do site, configuração da aplicação, cor de destaque ao tocar, cor do mosaico e imagem do mosaico para que tenha bom aspeto em dispositivos Windows.
- **Meta-tags Open Graph/Facebook:** Estas meta tags permitem-lhe controlar a forma como o seu sítio surge quando partilhado no Facebook e noutras plataformas de redes sociais. Ao definir o título, descrição e imagem, pode garantir que o seu sítio fica com o melhor aspeto quando partilhado online.
- **Barra de navegação responsiva:** A barra de navegação responsiva oferece aos utilizadores uma interface intuitiva e fácil de usar para navegar no sítio. Procura adaptar-se ao tamanho do ecrã, tornando-se acessível a utilizadores em dispositivos de secretária e móveis.
- **Meta tags Schema.org:** Estas meta tags são usadas para fornecer dados estruturados sobre o conteúdo do sítio. Definir o nome, descrição e imagem num sítio ajuda os motores de busca e outros a compreender melhor o conteúdo.
- **Meta tags Twitter:** Estas meta tags foram concebidas para otimizar o sítio para partilha no Twitter. Pode definir o tipo de cartão, criador, descrição, imagem, sítio, título e URL para que o seu sítio tenha bom aspeto no Twitter.
