## 入门模板

所有入门模板,无论是页面还是完整网站,都基于相同的理念和功能从零开始构建。
这意味着您可以轻松地将它们混合搭配,创建出符合自己需求的自定义布局和内容。

无论您想要搭建网站还是 Web 应用,都可以使用我们提供的任一入门模板,
快速开始 SSG 的使用之旅。

**Kaishi 入门模板** 启用了以下功能:

- **无障碍元标签:** 这些元标签旨在让网站对残障用户更加友好。通过设置无障碍富互联网应用(ARIA)角色与属性、完整的键盘控制以及避免闪烁隐患,您可以确保网站对所有人都是可访问的。
- **Apple 元标签:** 这些元标签为 iPhone、iPad 等 Apple 设备优化网站显示。您可以调整 Web 应用能力、状态栏样式、标题、应用名称和作者等信息,从而提升在 Apple 设备上的展示效果。
- **Bootstrap CSS:** Bootstrap 是一个流行的 CSS 框架,提供一整套预设的样式与组件。借助 Bootstrap,您无需从零编写 CSS,即可快速创建出专业外观的网站。
- **Bootstrap JavaScript:** Bootstrap JavaScript 是一组预构建的脚本,为您提供响应式导航菜单和模态对话框等功能。
- **内容安全策略:** 此元标签用于指定页面上允许加载的内容来源,旨在防范跨站脚本攻击(XSS)等安全漏洞。
- **微软元标签:** 这些元标签用于为微软设备优化网站显示。您可以设置站点验证、应用配置、点击高亮颜色、磁贴颜色和磁贴图片,使网站在 Windows 设备上获得良好显示效果。
- **Open Graph / Facebook 元标签:** 这些元标签让您可以控制网站在 Facebook 及其他社交媒体平台上被分享时的呈现方式。通过设置标题、描述和图片,您可以确保网站在分享时呈现出最佳状态。
- **响应式导航栏:** 响应式导航栏为用户提供直观易用的导航界面。它会根据屏幕尺寸自适应调整,让桌面和移动端用户都能顺畅访问。
- **Schema.org 元标签:** 这些元标签用于为网站内容提供结构化数据。设置网站的名称、描述和图片,有助于搜索引擎和其他服务更好地理解网站内容。
- **Twitter 元标签:** 这些元标签专为在 Twitter 上的分享优化网站展示。您可以设置卡片类型、创建者、描述、图片、站点、标题和 URL,让网站在 Twitter 上呈现出良好的效果。
