## Startovací šablony

Všechny startovací šablony, ať už stránky nebo celé weby, jsou postaveny od základů
za použití stejných principů a funkcí. To znamená, že je můžete snadno kombinovat
a vytvářet vlastní rozvržení a obsah.

Začněte se SSG pomocí kterékoliv z našich startovacích šablon pro tvorbu
vašeho webu nebo webové aplikace.

**Startovací šablony Kaishi** mají zapnuté následující funkce:

- **Meta značky pro přístupnost:** Tyto meta značky jsou navrženy tak, aby byl web přístupnější uživatelům se zdravotním postižením. Nastavením rolí a atributů ARIA (Accessible Rich Internet Applications), plné ovládání klávesnicí a bez rizika blikání zajistíte, že váš web bude přístupný všem.
- **Apple meta značky:** Tyto meta značky vylepšují weby pro zařízení Apple, jako jsou iPhony, iPady a další zařízení Apple. Můžete změnit možnosti webové aplikace, styl stavového řádku, titulek, název aplikace a autora, abyste vylepšili vzhled na zařízeních Apple.
- **Bootstrap CSS:** Bootstrap je populární CSS framework, který vám poskytuje sadu předpřipravených stylů a komponent. Díky Bootstrapu můžete rychle a snadno vytvořit profesionálně vypadající web, aniž byste museli psát CSS od nuly.
- **Bootstrap JavaScript:** Bootstrap JavaScript je sada předpřipravených skriptů, které poskytují responzivní navigační nabídky a modální dialogy.
- **Content Security Policy:** Tato meta značka se používá k určení zdrojů obsahu povolených k načtení na stránce. Je navržena tak, aby zabránila útokům cross-site scripting (XSS) a dalším bezpečnostním zranitelnostem.
- **Microsoft meta značky:** Tyto meta značky jsou navrženy tak, aby optimalizovaly web pro zařízení Microsoft. Můžete nastavit ověření webu, konfiguraci aplikace, barvu zvýraznění při klepnutí, barvu dlaždice a obrázek dlaždice, aby web dobře vypadal na zařízeních se systémem Windows.
- **Open Graph / Facebook meta značky:** Tyto meta značky vám umožňují ovládat, jak se váš web zobrazuje při sdílení na Facebooku a dalších sociálních sítích. Nastavením titulku, popisu a obrázku zajistíte, že váš web bude při sdílení online vypadat co nejlépe.
- **Responzivní navigační lišta:** Responzivní navigační lišta poskytuje uživatelům intuitivní a snadno použitelné rozhraní pro navigaci na webu. Jejím cílem je přizpůsobit se velikosti obrazovky, aby byla dostupná uživatelům na stolních i mobilních zařízeních.
- **Schema.org meta značky:** Tyto meta značky se používají k poskytování strukturovaných dat o obsahu webu. Nastavení názvu, popisu a obrázku na webu pomáhá vyhledávačům a dalším službám lépe porozumět obsahu.
- **Twitter meta značky:** Tyto meta značky jsou navrženy tak, aby optimalizovaly web pro sdílení na Twitteru. Můžete nastavit typ karty, tvůrce, popis, obrázek, web, titulek a URL, aby web na Twitteru vypadal dobře.
