## Plantillas de inicio

Todas las plantillas de inicio, ya sean páginas o sitios web, están construidas
desde cero usando los mismos principios y funcionalidades. Esto significa que
puedes mezclarlas y combinarlas fácilmente para crear tus propios diseños y
contenidos personalizados.

Empieza con SSG usando cualquiera de nuestras plantillas de inicio para construir
tu sitio web o aplicación web.

**Las plantillas de inicio Kaishi** tienen activadas las siguientes funcionalidades:

- **Metaetiquetas de accesibilidad:** Estas metaetiquetas están diseñadas para hacer el sitio web más accesible a usuarios con discapacidades. Al definir roles y atributos ARIA (Accessible Rich Internet Applications), control completo por teclado y ausencia de riesgos por destellos, te aseguras de que tu sitio sea accesible para todos.
- **Metaetiquetas de Apple:** Estas metaetiquetas mejoran los sitios web para dispositivos Apple, como iPhones, iPads y otros dispositivos Apple. Puedes cambiar las capacidades de la aplicación web, el estilo de la barra de estado, el título, el nombre de la aplicación y el autor para mejorar la apariencia en dispositivos Apple.
- **Bootstrap CSS:** Bootstrap es un popular framework CSS que te proporciona un conjunto de estilos y componentes prediseñados. Usando Bootstrap, puedes crear rápida y fácilmente un sitio web de aspecto profesional sin tener que escribir CSS desde cero.
- **Bootstrap JavaScript:** Bootstrap JavaScript es un conjunto de scripts predefinidos que te ofrece menús de navegación adaptables y diálogos modales.
- **Política de seguridad de contenido:** Esta metaetiqueta se usa para especificar las fuentes de contenido permitidas para cargarse en la página. Está diseñada para prevenir ataques de cross-site scripting (XSS) y otras vulnerabilidades de seguridad.
- **Metaetiquetas de Microsoft:** Estas metaetiquetas están diseñadas para optimizar el sitio web para dispositivos Microsoft. Puedes establecer la verificación del sitio, la configuración de la aplicación, el color de resaltado al pulsar, el color del mosaico y la imagen del mosaico para que se vea bien en dispositivos Windows.
- **Metaetiquetas de Open Graph/Facebook:** Estas metaetiquetas te permiten controlar cómo aparece tu sitio web cuando se comparte en Facebook y otras plataformas de redes sociales. Estableciendo el título, la descripción y la imagen, puedes asegurarte de que tu sitio luzca lo mejor posible cuando se comparta en línea.
- **Barra de navegación adaptable:** La barra de navegación adaptable proporciona a los usuarios una interfaz intuitiva y fácil de usar para navegar por el sitio web. Su objetivo es adaptarse al tamaño de la pantalla, haciéndola accesible para usuarios tanto en dispositivos de escritorio como móviles.
- **Metaetiquetas Schema.org:** Estas metaetiquetas se usan para proporcionar datos estructurados sobre el contenido del sitio web. Establecer el nombre, la descripción y la imagen en un sitio web ayuda a los motores de búsqueda y a otros a comprender mejor el contenido.
- **Metaetiquetas de Twitter:** Estas metaetiquetas están diseñadas para optimizar el sitio web para compartir en Twitter. Puedes establecer el tipo de tarjeta, el creador, la descripción, la imagen, el sitio, el título y la URL para que tu sitio web luzca bien en Twitter.
